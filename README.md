# homebrew-tmux-mcp

Homebrew tap for tmux-mcp-rs.

## Install

```bash
brew install bnomei/tmux-mcp/tmux-mcp-rs
```

## Maintain

Bump `version` and update all `sha256` values in `Formula/tmux-mcp-rs.rb` to match the
GitHub Release tarballs from `bnomei/tmux-mcp`.
